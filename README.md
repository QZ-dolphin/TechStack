# 个人技术栈前端展示页面
![demo](image.png)

## 初始化安装
```sh
npm install
```

### 运行
Compile and Hot-Reload for Development
```sh
npm run dev
```
### 同步脚本
linux环境下运行gitpush.sh
```bash
./gitpush.sh
```
### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
